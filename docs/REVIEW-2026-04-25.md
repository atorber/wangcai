# 旺财 App 界面 Review 报告

> 时间：2026-04-25  
> 范围：`lib/` 下 5 个 Tab 页面、设置页各子页面、Provider 数据层、WebDAV 备份服务、导出服务  
> 目的：识别未实现 / 假数据 / 交互断点，给出按优先级的改进方案与落地路线

---

## 一、整体架构概览

项目采用 **Flutter + Provider** 架构，底部 5 个 Tab：首页（资产概览）、统计、添加、账单、设置。

- 数据通过 `SharedPreferences` 本地持久化
- 支持 WebDAV 备份恢复 / JSON、CSV 导出
- 记账闭环（支出、收入、转账、借出、借入）已打通
- Provider：`AccountProvider` / `TransactionProvider` / `CategoryProvider` / `SecurityProvider` 均已本地化存储
- 记账会同步调整账户与借贷人余额，并支持回滚与更新

整体"可跑起来"的原型已成型，但**统计相关页面仍大量依赖 mock 数据**，是当前最显眼的缺口。

## 三、优先级改进方案

### 🚨 Phase 0：数据真实性修复（1~2 天）

统计是用户核心入口，当前是假的，必须最高优先级。

1. 新建 `lib/services/stats_service.dart`（或 Provider selector）
   - 输入：`List<TransactionRecord>` + period (周/月/年)
   - 输出：分类支出分布、日/周/月累计、同比环比、日均
2. 重写 `FinancialStatsScreen`，基于真实数据渲染饼图/柱状图，删除 mock 段；周/月/年切换接入真实逻辑
3. 修复 `MonthlyOverviewScreen._monthlyTotal`，按"当前月"过滤；标题改成动态 `${month}月概览`
4. 决定 `MonthlyOverviewScreen` 归属：
   - 方案 A：删掉（`AssetOverviewScreen` 已足够）
   - 方案 B：作为首页二级入口（tab 内切换），并接入导航
5. 删除 `github_sync_settings_screen.dart`（死代码）
6. 移除所有 Google 外链占位头像，改本地图标

### 🛠 Phase 1：承诺功能兑现（3~5 天）

7. **应用锁真正生效**：`AppLockGate` 包裹 `MyApp`，冷启动/从后台恢复时触发生物识别；接入 `local_auth` 支持 FaceID/指纹
8. **隐私模式**：封装 `PrivacyText` 组件，根据 `privacyModeEnabled` 渲染 `¥****`；全局金额替换
9. **主题切换**：拆 `AppSettingsProvider` 持久化 `themeMode`；补 `AppTheme.darkTheme`；设置页弹选择窗
10. **关于页**：新建 `AboutScreen`，读 `package_info_plus` 显示版本号、GitHub、开源协议
11. **移除"退出登录"**，或改为"清空本地数据"二次确认
12. **用户头像与昵称**：改为本地可编辑的 Profile，去掉假数据
13. **WebDAV 可靠性**：
    - 请求加 timeout 15s
    - 上传失败按类型提示（401/404/5xx）
    - 上传前 MKCOL 确保目录存在
    - bundle 增加 `schemaVersion` / `deviceId`；下载比较 `exportedAt` 做冲突提示
14. **密码迁移到 `flutter_secure_storage`**
15. **导入数据**：设置页新增"从文件导入 JSON"，复用 `AppBackupBundle.fromJson` + `replaceAll`

### 🎯 Phase 2：记账效率与统计深化（3~5 天）

16. **账单筛选**：`BillListScreen` 顶部 Chip 过滤（全部/支出/收入/转账/借贷）+ 日期范围 + 关键字搜索
17. **账单真分页**：`TransactionProvider.queryPage(filter)` 避免全量排序
18. **智能置顶 & 最近输入记忆**：`CategoryProvider` 持久化最近使用时间戳，按 recency 排序
19. **快捷金额模板**：金额输入框上方加 `+10 / +50 / +100 / 清零` 按键
20. **账户/借贷人详情页统计区块**：累计收支、月度趋势小卡片
21. **预算功能**：`Budget { categoryId, monthlyLimit }`，统计页展示执行进度
22. **保存账单后自动返回/跳转账单列表**

### 🌱 Phase 3：长期扩展

23. **多账本**：`Ledger` 模型，Provider 全部按 `ledgerId` scope
24. **自动规则 / 周期账单**：`RecurringRule` 模型，定时触发
25. **OCR / 分享记账**：从系统分享菜单识别金额
26. **测试覆盖**：记账、账户、WebDAV 上传/下载、分类重命名 4 个核心测试
27. **CI 质量门禁**：GitHub Actions 跑 `flutter analyze` + `flutter test`
28. **文件/类重命名**：`github_sync_*` → `webdav_backup_*`；`finance_app` → `wangcai`
