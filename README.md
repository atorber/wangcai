# 旺财 (WangCai)

**旺财** 取“兴旺发财”之意，是一款设计精美、功能实用的个人记账手机应用。使用 Flutter 构建，旨在帮助您轻松管理个人财务、追踪支出并实现储蓄目标，祝您财源广进。

## 使用说明 (User Guide)

以下是“旺财”APP的核心使用指南，帮助您快速上手：

### 1. 记一笔 (添加账单)
*   **入口**: 在应用底部的导航栏中，点击中间醒目的 **“添加” (+号)** 按钮。
*   **操作**:
    *   在弹出的面板顶部选择类型：`支出`、`收入`、`转账`、`借出` 或 `借入`。
    *   输入金额。
    *   在下方的分类网格中选择对应的消费类别（如：餐饮、交通、购物等）。
    *   可选：修改日期和添加备注（例如：“和朋友聚餐”）。
    *   点击底部的 **“保存账单”**。

### 2. 查看资产与账户
*   **入口**: 底部导航栏点击 **“首页”**。
*   **操作**:
    *   **资产概览**: 页面顶部会显示您的【总资产】、【净资产】和【总负债】。
    *   **我的账户**: 向下滚动可以查看所有已绑定的账户列表（包含储蓄卡、信用卡、支付宝、微信等）。点击卡片可以查看详细余额或信用卡本期应还金额。
    *   **添加新账户**: 点击列表底部的 **“添加账户”** 按钮，输入初始余额、账户名称并选择类型即可新增账户。

### 3. 查看财务统计与月度报告
*   **入口**: 底部导航栏点击 **“统计”**。
*   **操作**:
    *   **时间维度**: 顶部可切换【周】、【月】、【年】视图。
    *   **支出分布**: 环形图展示了各类别的支出占比（如住房、餐饮等）。
    *   **支出趋势**: 柱状图展示了近期的每日支出趋势，并与历史平均水平进行对比。
    *   **财务洞察**: 底部会提供智能提示，例如“预计可节省”金额或“支出超预算”警告。

### 4. 设置与数据安全 (GitHub 同步)
*   **入口**: 底部导航栏点击 **“设置”**。
*   **操作**:
    *   **个性化**: 调整应用主题（深色/浅色模式），管理分类等。
    *   **安全**: 开启面容ID/密码保护。
    *   **数据云同步**:
        1. 在设置中点击 **“同步到 GitHub”**。
        2. 按照向导输入具有相应权限的 **GitHub Personal Access Token (PAT)**。
        3. 选择需要同步的 **目标仓库** 和 **文件格式 (JSON/CSV)**。
        4. 配置同步频率（推荐“实时同步”或“每日同步”）。配置完成后，您的记账数据将安全地备份至您的私有 GitHub 仓库中，永不丢失。

## 功能特性

旺财目前已实现以下核心功能界面的构建：

1.  **资产概览 (`AssetOverviewScreen`)**: 全面掌握您的财务状况。
2.  **便捷记账 (`AddTransactionScreen`)**: 随时随地记录收支，便捷的模态弹出设计。
3.  **财务统计 (`FinancialStatsScreen`)**: 洞察您的消费习惯。
4.  **账户管理 (`AddAccountScreen`)**: 轻松添加和管理各类财务账户。
5.  **月度报告 (`MonthlyOverviewScreen`)**: 每月财务情况总结。
6.  **个性化设置 (`SettingsScreen`)**: 定制应用体验及安全选项。
7.  **数据云同步 (GitHub Sync)**: 完整的 GitHub 数据备份与同步设置流程。

## 技术栈

*   **跨平台框架:** Flutter
*   **字体设计:** `google_fonts` (采用现代化无衬线字体 Inter)
*   **数据可视化:** `fl_chart` (用于生成精美的统计图表)
*   **图标库:** `material_symbols_icons`
*   **状态管理:** `provider` (为后续业务逻辑处理和数据状态管理提供基础)

## 快速开始 (供开发者体验)

如果您想在本地运行和体验“旺财”应用的 UI：

1.  **准备工作:** 确保您的计算机上已安装并配置好 Flutter SDK 开发环境。
2.  **克隆仓库:**
    ```bash
    git clone <repository-url>
    cd finance_app
    ```
3.  **获取依赖:**
    ```bash
    flutter pub get
    ```
4.  **运行应用:**
    ```bash
    flutter run
    ```
